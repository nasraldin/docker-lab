# Screenshots & diagrams

Two kinds of visuals:

1. **Diagrams** (already in the repo) — install flow, stack, roadmap, release channels
2. **CLI / UI screenshots** — real terminal and browser captures for releases

## Diagrams

Generated by `scripts/generate-diagrams.py` (SVG source + PNG export via `rsvg-convert`).

| File | Used on |
| --- | --- |
| [`install-flow.png`](../assets/diagrams/install-flow.png) | Home, Installation, README |
| [`install-steps.png`](../assets/diagrams/install-steps.png) | Architecture |
| [`stack.png`](../assets/diagrams/stack.png) | Architecture, README |
| [`roadmap.png`](../assets/diagrams/roadmap.png) | Roadmap, README |
| [`release-channels.png`](../assets/diagrams/release-channels.png) | Releasing |

Regenerate after editing the script:

```bash
brew install librsvg   # once, for rsvg-convert
make docs-diagrams
```
